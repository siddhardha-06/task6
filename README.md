# Task 6 — Contact Form (Client-side validation)

Files added:

- `index.html` — the contact form UI
- `styles.css` — styling for form and messages
- `script.js` — validation logic and user feedback

How to run:

1. Open `index.html` in your browser (Chrome recommended).
2. Fill Name, Email, Message and click "Send Message".
3. Field-level errors appear if validation fails. On success a message shows.

Optional: use a static server (e.g., VS Code Live Server) to serve the page.
